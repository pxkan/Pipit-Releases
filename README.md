# Pipit Releases

This repository hosts public releases for **Pipit**, a voice-to-text transcription app for macOS.

## About Pipit

Pipit provides:
- Speech-to-text transcription with support for 25 languages
- Transcription history management
- Clipboard monitoring
- Optional AI post-processing
- Automatic updates via Sparkle

## Downloads

Visit the [Releases](https://github.com/pxkan/Pipit-Releases/releases) page to download the latest version.

## System Requirements

- macOS 14.0 (Sonoma) or later
- Apple Silicon or Intel processor
